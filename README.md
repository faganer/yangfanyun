1. 全节点均解锁主流流媒体与 AI 工具。  
2. 支持多平台订阅：Clash、Surge、Shadowrocket、Stash、sing-box 等。  
3. 支付宝直付，符合国区支付习惯。  
4. 官网后端及面板加载迅速，新人上手门槛低。  

### 不足
1. 官网附带的旧版 Clash 客户端存在误报风险，建议自行前往 GitHub 下载。  
2. 节点数目较少，网络高峰时段偶尔会有轻微抖动。  
3. 暂不支持按量付费或转赠流量。
